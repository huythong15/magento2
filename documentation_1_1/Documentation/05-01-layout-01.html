Documentation

Fashion

To make the theme look like this style:

  1. Active EM0146 Neccessary Fashion. Go to Active Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM Neccessary Homepage Two Columns Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

Horizontal Menu

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Main Megamenu (Identifier em0146-fashion-main-megamenu).

In content of block, you double click on Show/Hide Editor button to edit content as HTML tags

2. Main Content

Back to top

Advertisement above Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Home - Ads Text (Identifier em0146-fashion-home-ads-text).

Main Slideshow and Left Menu

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Area Main - Slideshow (Identifier em0146-fashion-main-area-main-slideshow).

Three Banners Below Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Area Main 01 - Home - Banners (Identifier em0146-fashion-area-main01-home-banners).

New Arrivals Tabs

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Area Main 02 - Home - Tabs (Identifier em0146-fashion-area-main02-home-tabs).

In content of block, you can edit Title of each tab

Content of each tab is EM Filter Product widget, you can double click on these widgets to edit widget options

If you would like to add more tabs, you need to click on Show/Hide Editorbutton to edit content as HTML tags and add tab as built-in structure

Fullwidth Banner

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Area Main 03 - Home - Banner (Identifier em0146-fashion-area-main03-home-banner).

Best sales

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Area Main 04 - Home - Best Sale (Identifier em0146-fashion-area-main02-home-best-sale).

In content of block, you double click on widget to configure EM Filter Products Widget

Please go to EM Filter Products Widget to learn more about how to manage this widget.

Advertisement

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Main Bottom 02 - Ads (Identifier em0146-fashion-main-bottom02-ads).

Brands Slider

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Main Bottom 03 - Banner Slider (Identifier em0146-fashion-main-bottom03-banners-slider).

3. Footer

Back to top

Footer Links - Newsletter - Social Icons

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Before Footer 01 - Links (Identifier em0146-fashion-before-footer01-links-).

In content of this block, it includes code to show Newsletter, static links and social icons

Payment Logos

In admin panel, select Content > Blocks, look for block with name EM0146 Neccessary - Fashion Store - Footer Inner - Payment (Identifier em0146-fashion-area-footer-inner-payment).

To change position of above blocks, you go to Content > Widgets, look for widget instance same name and edit Layout Update of widget instance.
